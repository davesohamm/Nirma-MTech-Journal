\#### üîπ Flask & Backend Concepts

\- \*\*Flask\*\* ‚Üí Lightweight Python web framework for building backend applications.

\- \*\*Jinja2\*\* ‚Üí Template engine used to dynamically render HTML pages.

\#### üîπ \`request\` Object in Flask

\- \`request.method\` ‚Üí HTTP method used (GET, POST, etc.).

\- \`request.args\` ‚Üí URL query parameters (GET data).

\- \`request.form\` ‚Üí Data submitted through HTML forms (POST).

\- \`request.files\` ‚Üí Uploaded files from client.

\- \`request.json\` ‚Üí JSON payload sent in request body.

\- \`request.headers\` ‚Üí Metadata of HTTP request.

\- \`request.cookies\` ‚Üí Cookies sent by client browser.

\- \`request.path\` ‚Üí URL path accessed.

\- \`request.full\_path\` ‚Üí Full path including query string.

\- \`request.url\` ‚Üí Complete request URL.

\- \`request.base\_url\` ‚Üí Base URL without query parameters.

\- \`request.remote\_addr\` ‚Üí Client IP address.

\- \`request.user\_agent\` ‚Üí Browser/system details of client.

\- \`request.data\` ‚Üí Raw request body data.

\- \`request.mimetype\` ‚Üí Type of content sent (e.g., application/json).

\#### üîπ Model Serialization & Execution Flow

\- \*\*Pickle\*\* ‚Üí Used to serialize (save) trained ML models.

\- \*\*Pickling a model\*\* ‚Üí Saving trained model into \`.pkl\` file.

\- \*\*Deserialization (\`rb\` mode)\*\* ‚Üí Loading model back using read-binary mode.

\- \*\*\`with open()\`\*\* ‚Üí Safe file handling; automatically closes file.

\- \*\*\`\_\_name\_\_\` and \`\_\_main\_\_\`\*\* ‚Üí Ensures app runs only when file is executed directly.

\- \*\*Route decorators (\`@app.route\`)\*\* ‚Üí Map URLs to Python functions.

\- \*\*\`render\_template()\`\*\* ‚Üí Renders HTML files using Jinja2.
